USAGE INSTRUCTIONS
================================================================================

1. Login as admin user
2. Click "Brands" in navigation menu
3. Click "Create Brand" button
4. Fill in brand name (required) and description (optional)
5. Check/uncheck "Active" checkbox
6. Click "Create" button

System automatically generates:
  - Unique QR code (e.g., AB12XYZ789)
  - QR image file (SVG)
  - SEO-friendly slug (e.g., test-restaurant)

To edit: Click "Edit" → Update → Save
To delete: Click "Delete" → Confirm
To view QR: Click "Edit" → Scroll to QR Code section

DEPLOYMENT CHECKLIST
================================================================================

Before deploying to production:

  □ Run composer install --optimize-autoloader --no-dev
  □ Run php artisan config:cache
  □ Run php artisan route:cache
  □ Run php artisan view:cache
  □ Run php artisan storage:link (if not exists)
  □ Set APP_ENV=production in .env
  □ Set APP_DEBUG=false in .env
  □ Configure database credentials
  □ Set up QR image backup strategy
  □ Set proper file permissions (755 directories, 644 files)
  □ Test QR codes on real devices
  □ Verify admin access control
  □ Check storage disk space
